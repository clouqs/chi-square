# chi-square
Chi-square test for estimating variance confidence interval given sample variance s^2, sample size and confidence level for HP Prime G2


# To import to calculator:

1) Copy the whole program
2) In the HP Connectivity Kit go to Programs (under your calculator's name), make a new file
3) Paste the program
4) Send to Class

5) done


Also has t-estimates and F-distribution for ratios
